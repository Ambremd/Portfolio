HTML / CSS - projet de cours

Développement d’un site à partir d’une maquette pour un constructeur de maison scandinave

Maquette initiale avec découpage

Contexte

- Projet réalisé en solo dans le cadre d’un exercice noté en 2ᵉ année de Bachelor Web & Multimédia.

- Durée : 1 mois.

- Objectifs : Mettre en pratique le responsive design, Git, GitHub et LESS.

- Contrainte : Reproduire strictement une maquette initiale en utilisant uniquement HTML et CSS tout en respectant des critères d’évaluation.

Problématique

Transformer une maquette graphique en un site fonctionnel.

Assurer la fidélité visuelle à la maquette tout en rendant le site adaptable à différents appareils (responsive design).

Méthodologie

  • Choix de la maquette sur Pinterest
  • Découpage de la maquette en sections
  • Structure HTML
  • Stylisation CSS
  • Responsive design
  • Intégration LESS
  • Utilisation de Git et GitHub

Outils utilisés

  • Visual Studio Code (écriture du code)
  • Git et GitHub (stockage et partage)

Langages utilisés

Code HTML

                    
<div class="maison" id="section-maison">
    <a href="maison1.html" class="maison1">
        <div class="overlay"></div>
        <h2 class="home">01</h2>
        <h3 class="bluesky">THE BLUE SKY HOME</h3>
        <p>Refuge Scandinave</p>
        <img src="./img/MaisonScandinave.jpeg" alt="Maison">
    </a>
</div>

Code CSS

.maison{
    height: 80vh;
    width: 100%;
    box-sizing: border-box;
    display: flex;
    flex-direction: row;
    margin-top: 20px;
    }
    
    .maison1, .maison2, .maison3, .maison4{
    width: 25%;
    position: relative;
    overflow: hidden;
    }
    
    .maison1:hover, .maison2:hover, .maison3:hover, .maison4:hover {
        transform: scale(1.02);
        z-index: 5;
    }
    .maison4:hover{
        margin-right: 4px;
    }
    
    /* filtre noir sur image*/
    .overlay {
        position: absolute; 
        top: 0; 
        left: 0; 
        width: 100%; 
        height: 100%; 
        background-color: rgba(0, 0, 0, 0.5); 
        z-index: 0; 
    }

Résultats obtenus

Le projet a respecté la maquette initiale tout en intégrant les critères demandés.
Ce projet a été noté 17,75 sur 20.

Ce projet m’a permis de découvrir l’utilisation de Git, que je ne connaissais pas auparavant, et d’apprendre à exploiter GitHub, une plateforme que je possédais mais que je n’avais encore jamais utilisée. J’ai également appris à mettre en place de nouveaux effets et à explorer de nouvelles techniques pour réaliser certains éléments, comme l’ajout d’une loupe ou l’intégration de vidéos.

Axes d’amélioration

  • Centralisation des styles dans un fichier CSS partagé pour éviter les doublons.
  • Personnalisation des titres et méta-descriptions pour améliorer le SEO.
  • Agrandissement du menu mobile pour une meilleure lisibilité.
  • Approfondissement de l’utilisation de LESS pour une implémentation complète.
