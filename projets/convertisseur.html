JavaScript / JSX - projet de cours

Convertisseur de prix, dollard en euros

Contexte

  • Projet réalisé en solo dans le cadre d’un TP de Développement front en 2ᵉ année de Bachelor Web & Multimédia.
  • Durée : 1 heure
  • Objectifs : Apprendre à développer une fonctionnalité simple en JavaScript, puis reproduire cette fonctionnalité en JSX pour comprendre la transition entre ces deux langages.
  • Contrainte : Suivre les consignes données par le formateur pour réaliser la fonctionnalité du plus simple au plus complexe.

Problématique

Comment développer une fonctionnalité de conversion de prix en respectant des consignes et en utilisant JavaScript ?

Comment exploiter les bases du JavaScript pour créer une fonctionnalité interactive, puis reproduire cette logique en JSX ?

Méthodologie

  • Reprise des bases de JavaScript vues l'année précédente.
  • Réalisation de la fonctionnalité étape par étape : formulaire simple, ajout de boutons, gestion des événements, conversion de devises, suppression de contenu.
  • Suivi des consignes pour reproduire chaque étape en JSX avec l’aide du formateur.
  • Utilisation de Vite pour la configuration et le développement du projet.

Outils utilisés

  • Visual Studio Code pour l’écriture du code.
  • Vite pour l’environnement de développement.

Langages utilisés

JavaScript

const Boutons = document.createElement("div");

const bouttonPlus = document.createElement("button");
bouttonPlus.innerText ='+1';
bouttonPlus.addEventListener('click', function() {
    euro.value = parseInt(euro.value) + 1;
    dollard.innerText = "Prix en dollard :" + (parseInt(euro.value) *1.05).toFixed(2);
})

const bouttonMoins = document.createElement("button");
bouttonMoins.innerText ='-1';
bouttonMoins.addEventListener('click', function() {
    euro.value = parseInt(euro.value) - 1;
    dollard.innerText = "Prix en dollard :" + (parseInt(euro.value) *1.05).toFixed(2);
})

document.body.appendChild(Boutons);
Boutons.appendChild(bouttonMoins);
Boutons.appendChild(bouttonPlus);

JSX

<div>
<button style={{ backgroundColor: 'red'}} onClick={handleClick}>- 1<button>
<button style={{ backgroundColor: 'green'}} onClick={handleClickoff}>+ 1<button>
</div>
