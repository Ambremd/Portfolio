Wordpress - projet de cours

Développement d’un site à partir d’une maquette sur un Coffee Shop : Café Délice

Contexte

  • Projet réalisé en solo dans le cadre d’un exercice noté en 2ᵉ année de Bachelor Web & Multimédia
  • Durée : 1 mois
  • Objectifs : Réaliser le site de notre choix à partir d'une maquette personnalisée conçue avec l'aide du formateur.
  • Contrainte : Créer une maquette, puis l'intégrer dans WordPress en utilisant un thème enfant, en respectant des critères d’évaluation comme la fidélité à la maquette, la propreté du code, l’utilisation de champs ACF, et la personnalisation via PHP.

Problématique

Comment transformer une maquette graphique en un site fonctionnel avec WordPress en utilisant un thème enfant, tout en respectant la fidélité à la maquette et en exploitant les fonctionnalités dynamiques offertes par WordPress ?

Maquette réalisée pour le projet

Méthodologie et outils utilisés

  • Création de la maquette avec Figma en suivant les consignes du brief.
  • Suivi des étapes : création du thème enfant, intégration HTML/CSS/PHP, utilisation des plugins WordPress.
  • Profiter des heures de cours pour anticiper les difficultés techniques.

Outils utilisés

  • Visual Studio Code pour l'écriture du code.
  • WAMP pour la gestion du serveur local.
  • WordPress pour l'intégration.
  • FileZilla pour la liaison entre le site local et le serveur distant.
  • Figma pour la réalisation de la maquette.

Langages utilisés

  • HTML

  • CSS

  • PHP

Code HTML avec PHP lié à WordPress afin d'intégrer les images et textes directement dans le CMS.

<div class="carte">
    <h2><?php the_field("nom_cafe"); ?></h2>
    <h3><?php the_field("titre_1"); ?></h3>
    <div class="aliment">
        <div class="boisson">
            <img src=<?php the_field("icone_1"); ?> alt="icone tasse à café">
            <p><?php the_field("nom_1"); ?></p>
        </div>
        <div class="boisson">
            <img src=<?php the_field("icone_2"); ?> alt="icone boisson chaude">
            <p><?php the_field("nom_2"); ?></p>
        </div>
        <div class="boisson">
            <img src=<?php the_field("icone_3"); ?> alt="icone boisson froide">
            <p><?php the_field("nom_3"); ?></p>
        </div>
        <div class="boisson">
            <img src=<?php the_field("icone_4"); ?> alt="icone alcool">
            <p><?php the_field("nom_4"); ?></p>
        </div>
        <div class="boisson">
            <img src=<?php the_field("icone_5"); ?> alt="icone gâteaux">
            <p><?php the_field("nom_5"); ?></p>
        </div>
    </div>
</div>

Récupérer les éléments réalisés dans WordPress et les intégrer dans la page (header, pied de page, bloc créé depuis WordPress).

<?php get_header(); ?>

<?php the_content(); ?>

<?php get_footer(); ?>

Résultats obtenus

Site fidèle à la maquette initiale avec des fonctionnalités dynamiques grâce à ACF.

Note obtenue : 15,5 pour la maquette et 17 pour l'intégration WordPress.

Ce projet m’a permis de découvrir la création de thème enfant, l’utilisation de champs ACF et l’exploitation de PHP pour personnaliser les contenus dynamiques. Il m’a également permis de mieux comprendre les possibilités de personnalisation offertes par WordPress.

Axes d’amélioration

  • Approfondir la personnalisation des templates avec PHP.
  • Utilisation plus poussée des champs ACF pour dynamiser davantage le contenu.
